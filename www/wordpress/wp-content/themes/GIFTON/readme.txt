GIFTON is a modern, clean, and highly customizable free eCommerce WordPress theme designed especially for online stores. Built for performance and flexibility, it is fully WooCommerce compatible and ready for your next project!

== Description ==

GIFTON is a lightweight and elegant eCommerce theme for WordPress, perfect for creating professional-looking online stores. It offers smooth WooCommerce integration, multiple homepage sections, and enhanced customization options like color control, typography, background images, and logo uploads. Designed with performance and style in mind, GIFTON suits small businesses, startups, and online shops wanting a simple yet premium presence.

Key Features:
* Modern, stylish design
* WooCommerce ready
* Customizable colors, typography, and backgrounds
* Upload your custom logo easily
* Responsive and mobile-friendly layout
* Optimized for speed and SEO
* Customizable homepage sections (Shop by Category, Featured Products, Best Sellers, Testimonials, Newsletter, Blog)
* Ready-to-use widget area (Sidebar)
* Translation ready

== Installation ==

1. Upload the theme files to the `/wp-content/themes/gifton` directory, or install the theme through the WordPress admin under Appearance > Themes > Add New.
2. Activate the theme through the 'Themes' menu in WordPress.
3. Customize your site by going to Appearance > Customize.

== Frequently Asked Questions ==

= Does GIFTON support WooCommerce? =
Yes, GIFTON is fully compatible with WooCommerce.

= Can I upload my own logo? =
Yes, go to Appearance > Customize > Site Identity to upload your custom logo.

= Are there color and typography options? =
Yes, GIFTON lets you customize primary colors and choose fonts from popular web-safe options.

= Is GIFTON translation ready? =
Yes, it is ready for translation and multilingual websites.
